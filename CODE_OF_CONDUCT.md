# Code of Conduct

This project is about building useful Bitcoin-on-Tails tooling.

Contributors are expected to:

- Stay focused on the project
- Be cool and technically honest
- Assume good faith when possible
- Avoid personal attacks, harassment, spam, or intentionally disruptive behavior
- Keep issues and pull requests relevant to improving the software

Strong opinions are welcome. Bad-faith behavior is not cool.

Maintainers may close, edit, or remove issues, pull requests, or comments that derail the project, attack other contributors, or create unnecessary noise.

Participation in this project is voluntary. The goal is simple: build better tools, review code honestly, and keep the signal high.
